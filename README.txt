Forum na projekt zespołowy.

Opis projektu: Stworzenie forum dyskusyjnego. Użytkownicy forum na podstawie swoich uprawnień mają dostęp do niektórych zamkniętych wątków. Wartościowe odpowiedzi użytkownika na pytania są nagradzane, rating użytkownika rośnie. Wątki można filtrować w zależności od tematyki. Wybrany stack technologiczny: Django dla backendu, React dla frontendu, MySQL dla bazy danych.
